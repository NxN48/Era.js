【一、首次启动】
（请确保您已经解压完所有文件！）
（请确保您已配置好 Python3 的开发环境！（记住务必勾选将Python路径加入到PATH）（有必要的时候请重启系统）（在命令提示符中输入 python 不报错））

2. 运行【!DEBUG-FRONT.bat】，直到游戏主窗口弹出并显示【Now, you can start the Game Script Engine.】
3. 运行【!DEBUG-BACK.bat】，然后就请尽情探索吧！
4. 游戏主窗口所显示的内容就是未来会在教程中学习的内容，所以进入到下一小节吧！
